# CLAWNET UI Breakdown

## Design System

### Visual Language
- **Theme**: Dark mode only (no light mode toggle for MVP)
- **Primary color**: Cyan/teal accent (#0ea5e9)
- **Background**: Near-black (#0a0a0b) with subtle gradients
- **Cards**: Elevated surfaces (#111113) with subtle borders
- **Text**: High contrast (#fafafa) with muted secondary (#a1a1aa)
- **Status colors**:
  - Success: Green (#22c55e)
  - Warning: Amber (#f59e0b)
  - Error: Red (#ef4444)
  - Info: Blue (#3b82f6)

### Typography
- **Headings**: Inter (system font fallback)
- **Body**: Inter
- **Monospace**: JetBrains Mono (for addresses, numbers, code)

### Component Principles
- No rounded corners > 8px (sharp, technical aesthetic)
- Subtle borders (1px, low opacity)
- No drop shadows (use border/glow effects sparingly)
- Grid-based layouts (12-column system)
- Consistent 4px spacing scale

## Responsive Strategy

| Breakpoint | Behavior |
|------------|----------|
| Desktop (1280px+) | Full layouts as designed |
| Tablet (768-1279px) | Collapse side panels, stack grids |
| Mobile (< 768px) | Single column, simplified navigation |

Priority for mobile:
1. Agent Dashboard (essential for monitoring)
2. Tasks Marketplace (core functionality)
3. Landing Page (conversion critical)
4. Leaderboard (read-only, simpler)
5. Economy Map (reduced to list view on mobile)
6. Deploy Agent (full flow required)
